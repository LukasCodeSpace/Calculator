# Calculator(Qt & C++)
A desktop calculator built using C++ and Qt Widgets.
The application replicates the basic functionality of the Windows calculator and was developed to practice GUI development and event-driven programming with Qt.

This project focuses on building a structured Qt application while keeping the code readable and maintainable.

## Features
- Two-display layout:
     - Upper display: Shows user input  
     - Lower display: Shows calculation results
- Buttons-based number and operator input
- Standard arithmetic operations
     - Addition
     - Subtraction
     - Multiplication
     - Division
- Power operator(^)

- Clear and backspace functionality
- Basic error handling(e.g. division by zero)

2. Open the project in **QT Creator**
3. Build and run
