# Calculator
A simple calculator using C++ and QT Widget. 
It works similar to the Windows calculator.

## Installation:
1. Clone the repository:  
     - git clone https://github.com/LukasCodeSpace/Calculator.git  
2. Open the project in **QT Creator**
3. Build and run

## Functions:
- Upper display: Displays the users input  
- Lower display: Displays the calculation results
- Buttons: Number and operator input
